# Slides Reader

Slides Reader 是一个AI集成的课件阅读器。你可以把 PDF、PPT 或 PPTX 课件放进应用中阅读，并让应用围绕课件内容生成课程简介、逐页讲稿、页面问答、练习试卷、错题本和阶段考试。

这个工具适合擅长通过课件和AI学习的人使用。
可以用于：
- 根据讲义自学课程
- 通过课件复习课程
- 生成练习巩固学习效果


## 项目功能

### 课件导入与阅读

- 支持导入 PDF、PPT 和 PPTX 课件。
- PPT 和 PPTX 会自动转换成可阅读的页面。
- 可以按页阅读课件，并查看每页对应的内容。
- 可以保留历史课件，后续继续阅读。

### 课程内容生成

- 根据整份课件生成课程简介。
- 为每一页课件生成适合讲解的逐页讲稿。
- 对某一页有疑问的课件生成单页讲稿
- 可以围绕当前页面提问，获得和页面内容相关的回答。
- 可以在阅读过程中补充图片附件，让问答参考更多上下文。

### 练习与考试

- 可以根据课件生成练习试卷。
- 可以完成答题并查看结果。
- 可以查看错题本，集中复习答错的问题。
- 可以选择多份课件生成阶段考试，用于综合复习。

### 设置与记录

- 可以在设置页填写自己的模型服务信息。
- 可以自定义生成课程简介、讲稿、问答和试卷时使用的提示词。
- 应用会保存课件、生成内容、答题记录和错题记录。

## 快速开始

下载最新release版。解压后，进入文件夹。然后双击：

```text
SlidesReader.exe
```

第一次启动可能需要等待一段时间。应用会自动准备运行环境，并在准备完成后打开浏览器。如果浏览器没有自动打开，请查看终端窗口中的地址，例如：

```text
Open: http://localhost:8000/
```

然后把这个地址复制到浏览器地址栏。

发行版必须保留整个文件夹，不要只复制单独的 `SlidesReader.exe`。推荐保持下面这样的结构：

```text
SlidesReader/
  SlidesReader.exe
  backend/
  frontend/
  runtime/
  tools/
  storage/
  README.md
```

如果电脑里已经有 LibreOffice，应用会直接使用它处理 PPT 和 PPTX。如果没有，可以手动下载，或是让启动脚本自动下载（可能不稳定）。LibreOffice 是一个办公套件，本项目用它把 PPT 和 PPTX 转成可以阅读的页面。

使用发行版时，运行数据默认保存在：

```text
SlidesReader/storage/
```

这里包含课件、生成内容、答题记录和日志。不要随意删除这个文件夹，否则历史数据会丢失。

停止应用时，回到启动时打开的终端窗口，按：

```text
Ctrl+C
```


## 开始使用

1. 启动应用并打开网页。
2. 进入设置页，填写模型服务地址、密钥和模型名称。
3. 回到文件页，上传 PDF、PPT 或 PPTX。
4. 等待课程简介和逐页讲稿生成。
5. 点击阅读，查看课件、讲稿和当前页问答。
6. 根据需要生成试卷、答题、查看结果和复习错题。



## 源码部署方法

如果你拿到的是源码版，请先确认电脑已经准备好运行源码所需的软件。然后在项目根目录打开终端。

Windows 用户运行：

```powershell
python start.py
```

macOS 或 Linux 用户运行：

```bash
python3 start.py
```

第一次启动会自动准备运行所需内容，可能需要几分钟。启动成功后，应用会自动打开浏览器。如果没有自动打开，请复制终端里的 `Open:` 地址到浏览器。

源码版也可以使用这些启动方式：

不自动打开浏览器：

```powershell
python start.py --no-open
```

只检查环境，不启动应用：

```powershell
python start.py --diagnostics
```

指定端口：

```powershell
python start.py --port 8010
```

如果启动失败，请先运行诊断命令，再查看：

```text
storage/logs/
```

这个目录里保存了启动日志和错误信息。
